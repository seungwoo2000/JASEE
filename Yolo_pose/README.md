# 앉은 자세 분류 모델 개발 프로젝트

## 프로젝트 개요
- **목적**: YOLOv8 + Attention MLP 기반 앉은 자세 GOOD/BAD 분류
- **데이터**: 1,199장
- **최종 모델**: Attention MLP
- **최종 Recall**: 0.9545

## 폴더 구조 설명
- **01_data_preprocessing**: 데이터 전처리 및 랜드마크 추출
- **02_model_comparison**: 머신러닝/딥러닝 모델 비교 실험
- **03_model_improvement**: 피처 엔지니어링 및 튜닝을 통한 성능 개선
- **04_final_model**: 최종 최적화 모델 저장 및 로드 코드
- **05_documentation**: 프로젝트 보고서 및 포트폴리오 문서

## 핵심 성과
- **10개 모델 비교 실험**을 통한 최적 알고리즘 선정
- **데이터 누수(Leakage)** 발견 및 완벽 해결 (정직한 성능 확보)
- **4단계 고도화**를 통해 Recall 0.95 및 과적합 방지 성공
