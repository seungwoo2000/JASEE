# Posture Analysis Model Evaluation & Optimization Report

본 보고서는 1,199장의 자세 데이터를 기반으로 한 머신러닝 및 딥러닝 모델의 성능 평가 및 최적화 과정을 정리한 결과입니다.

## 1. 프로젝트 개요
- **목표**: 데이터 누수가 없는 엄격한 환경에서 나쁜 자세(BAD) 탐지율(Recall)을 극대화한 최적 모델 도출
- **데이터셋**: 1,199장 (Train 64% / Val 16% / Test 20%)
- **평가 지표**: Recall(최우선), F1-Score, AUC, Accuracy, Train/Val Diff(과적합 여부)

## 2. 모델 평가 및 고도화 플로우

### [STEP 1] 초기 베이스라인 구축 (10개 모델 비교)
- 머신러닝(LogReg, DT, XGB, RF) 및 딥러닝(MLP, Attention, 1D-CNN, EfficientNet 등) 성능 비교
- **결과**: Attention MLP가 가장 높은 잠재력을 보임

### [STEP 2] 데이터 누수(Data Leakage) 완벽 제거
- **문제**: CVA, TIA 각도값이 입력 피처에 포함되어 성능이 비정상적으로 높게 나옴 (정답 유출)
- **조치**: 각도 및 정답 유출 가능성이 있는 모든 x좌표 차이 피처 제거
- **결과**: 실제 서비스 환경과 동일한 정직한 성능 지표 확보 (Recall 0.9182)

### [STEP 3] 피처 엔지니어링 (16개 피처 확정)
- 누수가 없는 기하학적 피처 추가
  - **관절 간 거리(4개)**: 귀-어깨, 어깨-골반, 골반-무릎, 무릎-발목
  - **관절 간 각도(2개)**: 어깨-골반-무릎, 골반-무릎-발목
- **효과**: 신체 구조적 특성을 반영하여 모델의 이해도 향상

### [STEP 4] 임계값(Threshold) 최적화
- 기본값 0.5에서 **0.40**으로 조정
- **효과**: 정밀도(Precision) 손실을 최소화하면서 **Recall을 약 4% 추가 향상**

### [STEP 5] 하이퍼파라미터 튜닝 (Optuna)
- **최적화 대상**: Learning Rate, Dropout, Layer 구조, Batch Size, Weight Decay
- **제약 조건**: Train/Val 성능 차이(Diff) 0.05 이내 엄격 제한
- **결과**: 가장 안정적이면서 높은 Recall을 내는 최적 조합 도출
