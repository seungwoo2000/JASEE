# 모델 성능 개선
- **개선 단계**:
  1. **임계값 최적화**: Recall 극대화를 위한 Threshold 조정 (0.5 -> 0.40)
  2. **피처 엔지니어링**: 데이터 누수 제거 및 물리 피처(거리, 각도) 추가
  3. **Optuna 튜닝**: 하이퍼파라미터 자동 탐색 및 과적합 제어
  4. **AdamW & Scheduler**: 학습 안정성 및 일반화 성능 강화
- **단계별 성능**:
| 단계 | Recall | F1 | Diff |
| :--- | :--- | :--- | :--- |
| STEP1 임계값 | 0.9636 | 0.9422 | 0.0418 |
| STEP2 피처 | 0.9182 | 0.9224 | 0.0496 |
| STEP3 Optuna | 0.9545 | 0.9318 | 0.0384 |
| STEP4 AdamW | 0.9364 | 0.9406 | 0.0052 |
